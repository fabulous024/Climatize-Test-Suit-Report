## Why Rova AI?
- AI-driven test creation

- Fast setup and execution

## Test Scenarios
1. Navigate to Project Details page
2. Unauthenticated user navigation

## Devices Used
- Google Pixel 5a 5G (12) 

# How to Run
## Prerequisites
-	Access to Rova AI platform (www.rova.qa)
-	Mobile app build (APK for Android / IPA for iOS)
-	Test environment configured
  ## Steps to Execute
- Navigate to www.rova.qa
- Select mobile Apps tab on right top header
- Log in to the Rova AI dashboard
- Upload or connect your mobile application build
- Select the test suite you want to run https://fabulous024.github.io/Climatize-Test-Suit-Report/
- Choose the target device 
- Click Run Test
- Monitor execution and review results in the dashboard
## Viewing Results
- Test results are available in the Rova AI reporting dashboard
- Includes:
-	Pass/Fail status
-	Execution logs
-	Screenshots/videos
-	Error insights
